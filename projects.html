Building

Research and open-source projects I am building.

loopcanary

Research · In progress ★ Code

An open-source Python library for agent-loop observability — it watches an agent's execution and diagnoses when the loop goes off-rails: the same tool call firing over and over, silent context compression, memory writes that never get recalled, cost run-away, or a safety monitor raising an alarm. Emits a per-run timeline you can inspect after the fact or stream live. (Grew out of a batch stress-test tool for AI safety monitors.)

coact

Eng · In progress ★ Code

A governance substrate for two coding agents in one repo — letting Claude Code and Codex work in the same project safely, coordinating through the filesystem instead of the token stream: capability policies, advisory locks and a shared task board, worktree isolation with merge gates, and an append-only journal for replayable oversight. Designed around three axes — security, controllability, and cost. A single dependency-free Go binary.

Where does an agent's capability come from?

Research · In progress

Current research on attributing LLM-agent capability — is a gain supplied by the model's weights, the runtime context, or the surrounding harness (planners, verifiers, retry logic)? The key test is cross-harness transfer: does a post-training gain survive when the model runs under a different harness? It's the proxy-fidelity question one level up — whether a reported "agent score" measures the model or the scaffolding around it.

Personal

Wealth Projection Simulator

Eng · Shipped ★ Live Code

A client-side net-worth projection simulator — model household net worth year by year from your own assumptions (careers, windfalls, major expenses, lifestyle creep, low/mid/high return bands) and goal-seek "what would it take to hit $X by age Y?". Runs entirely in the browser — no backend, strict CSP, nothing stored or sent anywhere. Something I built for myself.
