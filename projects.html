monitorstress

In progress Code

An adversarial stress-test suite for LLM-based safety monitors — the class of system labs like METR and UK AISI use to detect agent reward hacking and sandbagging. Built on a typed Python/Pydantic-v2 stack, it applies semantics-preserving transformations (chain-of-thought stripping, observation truncation, action-padding noise) to agent transcripts from METR's MALT split and measures how monitor discrimination degrades — removing a model's reasoning traces collapses a Claude-Sonnet-based monitor from AUROC 0.683 to chance.
